`argument-differ` denotes an issue with the [Liskov Substitution Principle](https://en.wikipedia.org/wiki/Liskov_substitution_principle).
This means that the code in question violates an important OOP design principle which does not have one single solution.
We recommend searching online for the best solution in your case.

To give some examples of potential solutions:

* Add the argument to the parent class
* Remove the inheritance completely
* Add default arguments to the child class
